# 🌸 Floramics - Amigos de las flores

## Descripción del Proyecto

**Floramics** es un sitio web estático desarrollado como ejercicio práctico de HTML y CSS. El proyecto consiste en una página web sencilla con temática floral, diseñada para poner en práctica los conocimientos adquiridos durante el ciclo formativo.

## 📚 Contexto Educativo

Este proyecto fue realizado como parte del aprendizaje en el **Ciclo Formativo de Desarrollo de Productos Editoriales Multimedia**.

- **Plataforma de aprendizaje:** [AulaClic](https://www.aulaclic.es/index.htm)
- **Fecha de realización:** Navidades del 2017
- **Tipo:** Ejercicio práctico / Proyecto de prácticas

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica del sitio
- **CSS3** - Estilos y diseño visual básico
- **Git** - Control de versiones
